Source.Utils.load_csv module¶

Created on Fri Jun 5 15:06:32 2020

@author: arsii

Load all .csv files in a given folder. The files are loaded as pandas dataframes which are stored in a ictionary. All the subfolders in the given folder are ignored. If there is file of a type other than csv in the folder, an exception is raised.
